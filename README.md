#  Statistic Calculator using NumPy

##  Overview
This project calculates statistical values like mean, median, variance, and more using Python and NumPy.

##  Features
* Mean
* Median
* Standard Deviation
* Variance
* Min, Max
* Range
* Sum

##  How to Run
Enter numbers like:
1 2 3 4 5
Press Enter

## Tech
Python, NumPy

## Sample Output
Enter the numbers:
1 12 34 2 1 0 9 40 3

Mean                : 11.3333
Median              : 3.0000
Standard Deviation  : 14.2984
Variance            : 204.4444
Minimum             : 0.0000
Maximum             : 40.0000
Range               : 40.0000
